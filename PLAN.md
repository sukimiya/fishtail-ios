# FishPal iOS — Update Plan

## Current State

- Tab 1: PredictView — 地图 + GPS + 搜索 + 评分环 + 24小时柱状图
- Tab 2: ShareView — 照片网格（仅本地）
- 暗色主题，无动画，UI较简陋

---

## Phase 1 — UI 全面升级

### 1.1 底部导航栏重设计
- 4个标签：预测、地图、记录、我的
- 图标使用 SF Symbols，选中时有高亮动画
- 背景使用毛玻璃效果（.ultraThinMaterial）

### 1.2 预测页（PredictView）重设计
- 顶部搜索栏固定，简洁
- 预测卡片浮层，圆角卡片带阴影
- 评分环更大更醒目，中间显示分数和label
- 8个环境因子改为横向滚动卡片（图标 + 数值）
- 24小时图表优化：高亮当前小时、tooltip

### 1.3 地图页独立
- 从 PredictView 剥离成单独 MapView 标签
- 地图全屏展示
- 底部弹出钓点列表（sheet）
- 点击钓点显示详情卡片

### 1.4 主题优化
- 统一圆角半径（12pt）
- 加入 haptic feedback
- 加载状态用 skeleton 占位

---

## Phase 2 — 新功能

### 2.1 钓点详情页
- 钓点名称、坐标、描述
- 物种列表（species caught）
- 点赞按钮（调用后端）
- 该钓点的渔获照片

### 2.2 添加钓点
- 长按地图选点 或 搜索地址
- 填写：名称、描述、鱼种
- 提交到后端

### 2.3 渔获记录（Log）
- 替换现有 ShareView
- 每条记录：照片、鱼种、重量、地点、时间、备注
- 本地存储（SwiftData 或 UserDefaults JSON）
- 列表展示 + 详情页

### 2.4 "我的"页面
- 显示昵称
- 历史预测查询记录
- 退出登录
- 版本信息

---

## Phase 3 — 体验优化

### 3.1 离线缓存
- 缓存最近一次预测结果
- 无网络时显示缓存数据并提示

### 3.2 好钓情提醒（本地通知）
- 用户可开启：当当前位置预测分数 > 阈值时推送通知
- 每天早晨定时检查
